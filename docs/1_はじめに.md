# はじめに　- 5 min

この教材は、Vanilla JS x ヘッドレスCMSでサイトを構築するステップが体験できる入門ガイドです。

## 前提知識

下記の内容を知っていると学習を進めやすいです。

- HTML/CSSを使ってサイトを構築したことがある
- JavaScriptの基礎を理解している（配列操作やDOM操作など）
- WordPressを使ってサイトを構築したことがある

## どんなことができるようになるか

- HTML／CSS／JSの基礎を活かしながら動的なサイトを作れるようになる
  - [microCMS](https://microcms.io/) から記事コンテンツを取得してきて、動的な情報をサイトへ埋め込むことができるようになる
- [Cloudflare Pages](https://www.cloudflare.com/ja-jp/developer-platform/products/pages/) を使って、手元のデータをインターネット上に公開できるようになる

## 学習の流れ

1. CMSついて理解する（`docs/2_CMSについて.md`）
2. JavaScript の基礎（API/非同期/配列・オブジェクト/DOM）を確認（`docs/3_Javascript基礎.md`）
3. microCMS の設定とデータ取得〜画面表示までを体験（`docs/4_microCMSを使ってみる.md`）
4. 完成したファイルを Cloudflare Pages で公開（`docs/5_サイトに公開しよう.md`）
5. ~~再利用しやすい形に一般化（関数化・描画差し替え・設定共通化）（`docs/他の案件で使用する場合.md`）~~
