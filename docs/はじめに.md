# はじめに　- 5 min

この教材は、はじめての人でも「CMSを使って構築した成果物を、インターネット上に公開する」までを、小さなステップで体験できる入門ガイドです。

## 取り扱うこと

- **CMSの基礎**（まずは全体像をつかむ）
- **JavaScript の入門**（必要なところだけ）: 
  - API ってなに？どうやってデータをもらうの？
  - `fetch()` と `async/await` の使い方
  - 配列・オブジェクトのカンタン操作と、画面（HTML）に出すやり方（`docs/Javascript基礎.md`）
- **microCMS を用いた構築**
- **サイトに公開する方法**

## どんなことができるようになるか

- microCMS からニュース記事をとってきて、一覧ページ・詳細ページに表示できるようになる
- `fetch()` と `async/await` を使って、通信を待ちながら安全に処理を進められるようになる
- ひとつ作った取得関数を、別ページや別案件でも「表示だけ差し替えて」使い回せるようになる
- Cloudflare Pages を使って、手元のデータをインターネットに公開できるようになる

## 学習の流れ

1. ヘッドレス CMS の基礎を把握する（`docs/CMSについて.md`）
2. JavaScript の基礎（API/非同期/配列・オブジェクト/DOM）を確認（`docs/Javascript基礎.md`）
3. microCMS の設定とデータ取得〜画面表示までを体験（`docs/microCMSを使ってみる.md`）
4. 完成したファイルを Cloudflare Pages で公開（`docs/サイトに公開しよう.md`）
5. 再利用しやすい形に一般化（関数化・描画差し替え・設定共通化）（`docs/他の案件で使用する場合.md`）
