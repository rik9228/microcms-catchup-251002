# はじめに　- 5 min

この教材は、「CMSを使って構築した成果物を、インターネット上に公開する」までを、<br/>小さなステップで体験できる入門ガイドです。

## 取り扱うこと

- **CMSの基礎**
- **JavaScript の入門**: 
  - API ってなに？どうやって外部データを取得するの？
  - `fetch()` と `async/await` の使い方
  - 配列・オブジェクトの操作と、画面（HTML）に出すやり方（`docs/Javascript基礎.md`）
- **Vanilla JS x microCMS を用いた構築**
- **サイトに公開する方法**

## 前提知識

下記の内容を知っていると学習を進めやすいです。

- HTML/CSSを使ってサイトを構築したことがある
- JavaScriptの基礎を理解している（配列操作やDOM操作など）
- WordPressを使ってサイトを構築したことがある

## どんなことができるようになるか

- microCMS から記事コンテンツを取得してきて、動的な情報をサイトへ埋め込むことができるようになる
- `fetch()` と `async/await` を使って、非同期処理を用いた通信処理を行えるようになる
- Cloudflare Pages を使って、手元のデータをインターネット上に公開できるようになる

## 学習の流れ

1. ヘッドレス CMS の基礎を把握する（`docs/CMSについて.md`）
2. JavaScript の基礎（API/非同期/配列・オブジェクト/DOM）を確認（`docs/Javascript基礎.md`）
3. microCMS の設定とデータ取得〜画面表示までを体験（`docs/microCMSを使ってみる.md`）
4. 完成したファイルを Cloudflare Pages で公開（`docs/サイトに公開しよう.md`）
5. 再利用しやすい形に一般化（関数化・描画差し替え・設定共通化）（`docs/他の案件で使用する場合.md`）
